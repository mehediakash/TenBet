Agents dashboard -----


An Agent is a person who:

Brings in new players/users

Provides or recharges balance for their gameplay

Earns profit from their own clients as well

Has a partnership model with the company

Main Responsibilities of Agents

Acquiring New Players
Agents create new users using their reference/ID.
The user account is added under the agent.

Managing User Deposits and Withdrawals
If a player doesn’t pay, the agent adds balance on their behalf.
If a player requests a withdrawal, the agent approves it or sends a request to the company.

User Balance Control
Agents can add or decrease user balances according to system permissions.
Agents can monitor how much a user is betting.

Commission / Profit Income
Agents typically earn:

Loss Commission: Profit when a user loses

Turnover Commission: A percentage of the user’s total bets

Profit Share: Based on the company-agent partnership ratio

Managing Their Team / Downline
An agent can have sub-agents under them.
Users are under sub-agents.
Multi-level structure example: A1 → A2 → A3 → User

Agent Wallet System:

Agents also have their own balance:

Agent → Receives balance from the Company (that's balance  give main admin)

Agent → Gives balance to the User

Agent Commission System
It will be calculated in the backend: this commission system will be dynamic and can be set from the admin panel.

User Linking System--
Master Agent
 └── Agent
       └── Sub-Agent
              └── Users

Downline agent commission split
Example:
Master Agent 40% → Agent 30% → Subagent 30%
